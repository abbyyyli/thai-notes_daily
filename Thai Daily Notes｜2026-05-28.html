ไม่ทัน mâi than not in time; can’t make it Practical Usage 時間上來不及；常接 ไป / bpai / go、ตอบ / dtɔ̀ɔp / reply。 ถ้าแวะหลายที่ อาจจะไปไม่ทัน
thâa wɛ́ lǎai thîi, àat jà bpai mâi than
If we stop by many places, we might not make it in time.
แวะ wɛ́ stop by; drop by Local usage 順路停一下，不是正式拜訪。曼谷生活超常用。 เดี๋ยวแวะยิมก่อนนะ
dǐaw wɛ́ yim gɔ̀ɔn ná
I’ll stop by the gym first.
เลือกไม่ถูก lʉ̂ak mâi thùuk can’t choose; don’t know which is right Practical Usage V + ไม่ถูก 表示不知道怎麼做才對、哪個方式合適。 เมนูเยอะจนเลือกไม่ถูก
mee-nuu yə́ jon lʉ̂ak mâi thùuk
There are so many menu items that I can’t choose.
ไม่ออก mâi ɔ̀ɔk can’t get it out; can’t figure it out Practical Usage V + ไม่ออก 表示想法、話、記憶、差異「出不來」。 คิดไม่ออกว่าจะกินอะไร
khít mâi ɔ̀ɔk wâa jà gin à-rai
I can’t think of what to eat.
จน jon until; so...that; to the point that Practical Usage 把前面的狀態推到後面的結果。可表示時間「直到」或程度「到……」。 ยุ่งจนกินข้าวไม่ทัน
yûng jon gin khâaw mâi than
I was so busy that I didn’t have time to eat.
งั้น ngán then; in that case Local usage 口語「那、那就、既然這樣」。常放句首，引出下一步決定。 งั้นเอาอันนี้ก็แล้วกัน
ngán ao an níi gɔ̂ɔ lɛ́ɛw gan
Then I’ll take this one.
เหมาะ mɔ̀ suitable; fit; appropriate Practical Usage 某選擇跟人、場合、條件、目的很搭。常接 กับ / gàp / with; for。 คำนี้ใช้กับเพื่อนได้ แต่ไม่เหมาะกับลูกค้า
kham níi chái gàp phʉ̂an dâai, dtɛ̀ɛ mâi mɔ̀ gàp lûuk-kháa
This word can be used with friends, but it’s not suitable for customers.
แรง rɛɛng strong; intense; harsh Local usage 可形容風雨、味道、藥效、語氣、反應很強。พูดแรง 不是大聲，而是話很重。 คำนี้แรงไป ใช้กับลูกค้าไม่เหมาะ
kham níi rɛɛng bpai, chái gàp lûuk-kháa mâi mɔ̀
This word is too strong. It’s not suitable to use with customers.
วันหยุดไปไหนมามั่ง wan-yùt bpai nǎi maa mâang Where did you go during the holiday? Local usage มา 表示去過回來；มั่ง 是 บ้าง 的口語版。 แล้วเธอล่ะ วันหยุดไปไหนมามั่ง
lɛ́ɛw thəə lâ, wan-yùt bpai nǎi maa mâang
What about you? Where did you go during the holiday?
ลืม...ไว้ที่... lʉʉm ... wái thîi ... left ... at ...; forgot ... at ... Practical Usage ไว้ 表示東西被留在那裡的結果感。交通工具上常用 บน。 ลืมกระเป๋าไว้ที่คาเฟ่ ต้องแวะกลับไปเอา
lʉʉm grà-bpǎo wái thîi gaa-fɛɛ, dtɔ̂ng wɛ́ glàp bpai ao
I left my bag at the café. I have to stop by and get it.
ยังไม่ครบ yang mâi khróp still not complete Practical Usage 該有的東西、人、資料、餐點還沒有全部齊全。 วันนี้คำศัพท์ยังไม่ครบ เดี๋ยวค่อยเก็บต่อ
wan-níi kham-sàp yang mâi khróp, dǐaw khɔ̂i gèp dtɔ̀ɔ
Today’s vocabulary isn’t complete yet. We’ll continue collecting it later.
ขาด khàat missing; lack; be short of Practical Usage 本來應該有，但少了某一部分。常和 ครบ 搭配。 ยังไม่ครบค่ะ ขาดอีกหนึ่งหน้า
yang mâi khróp khâ, khàat ìik nʉ̀ng nâa
It’s still not complete. One more page is missing.
กินไม่หมด gin mâi mòt can’t finish eating it Local usage 食物太多或已經飽了，所以沒辦法全部吃完。不是 กินไม่ได้。 กินไม่หมดค่ะ ขอห่อกลับบ้านได้ไหมคะ
gin mâi mòt khâ, khɔ̌ɔ hɔ̀ɔ glàp bâan dâai mái khá
I can’t finish it. Can I have it packed to go?
ที่ชาร์จ thîi châat charger Local usage 英文 charge 的借詞。可泛指充電器；更精準可分充電線與充電頭。 แบตจะหมดแล้ว ขอยืมที่ชาร์จหน่อยได้ไหมคะ
bɛ̀ɛt jà mòt lɛ́ɛw, khɔ̌ɔ yʉʉm thîi châat nɔ̀i dâai mái khá
My battery is about to die. Can I borrow a charger?
ยิม yim gym Local usage 英文 gym 的泰文音譯。Condo 設施也常說 ฟิตเนส / fít-nêt / fitness center。 เพิ่งกลับจากยิมค่ะ
phə̂ng glàp jàak yim khâ
I just came back from the gym.
ต่อ dtɔ̀ɔ continue; connect; transfer; extend Review 核心是「接下去、接上」。今天延伸到 ต่อวีซ่า / dtɔ̀ɔ wii-sâa / extend a visa。 วีซ่าใกล้หมดอายุแล้ว ต้องต่อวีซ่า
wii-sâa glâi mòt aa-yú lɛ́ɛw, dtɔ̂ng dtɔ̀ɔ wii-sâa
My visa is about to expire, so I need to extend it.
มีอายุ mii aa-yú be valid; have an age / duration Supplement 可表示人的年齡，也可表示簽證、護照、文件、優惠券的有效期。 วีซ่านี้มีอายุ 90 วันค่ะ
wii-sâa níi mii aa-yú gâo-sìp wan khâ
This visa is valid for 90 days.
ส่วน sùan part; portion; section; as for Practical Usage 表示整體中的某一部分，也可句首表示「至於……」。 ข้อมูลขาดบางส่วนค่ะ
khɔ̂ɔ-muun khàat baang sùan khâ
Some information is missing.
ตั้ง dtâng set up; establish; set; place Supplement 核心畫面是「把某物立起來、放定、設定好」。延伸成 ตั้งใจ、ตั้งคำถาม、ตั้งอยู่。 ขอโทษนะคะ ไม่ได้ตั้งใจ
khɔ̌ɔ-thôot ná khá, mâi dâai dtâng-jai
Sorry, I didn’t mean to.
เปลี่ยน bplìian change; switch; exchange Practical Usage 把原本的人、事、物、時間、地點、選項、狀態改成另一個。 ขอเปลี่ยนเวลาเป็นบ่ายสองได้ไหมคะ
khɔ̌ɔ bplìian wee-laa bpen bàai sɔ̌ɔng dâai mái khá
Can I change the time to 2 p.m.?
ไม่...นัก mâi ... nák not very...; not that... Supplement 降低程度的句型，比 ไม่ค่อย... 稍正式、理性。 คำนี้ไม่เหมาะนักถ้าใช้กับลูกค้า
kham níi mâi mɔ̀ nák thâa chái gàp lûuk-kháa
This word isn’t very suitable if used with customers.

1. ไม่ถูก vs ไม่ออก vs ไม่ได้ vs ไม่เป็น

核心差異：ไม่ถูก / mâi thùuk / not in the right way 是「不知道怎麼做才對」；ไม่ออก / mâi ɔ̀ɔk / can’t get it out 是「東西出不來」。
PatternEnglish中文語感例句
พูดไม่ถูก
phûut mâi thùuk		 don’t know how to say it properly 不知道怎麼說才合適 เรื่องนี้ซับซ้อนมาก พูดไม่ถูก
rʉ̂ang níi sáp-sɔ́ɔn mâak, phûut mâi thùuk
This matter is complicated. I don’t know how to explain it properly.
พูดไม่ออก
phûut mâi ɔ̀ɔk		 speechless 情緒太強，話出不來 ตกใจจนพูดไม่ออก
dtòk-jai jon phûut mâi ɔ̀ɔk
I was so shocked that I couldn’t speak.
พูดไม่ได้
phûut mâi dâai		 can’t speak 能力、規則、情況上不能說 ฉันพูดภาษาไทยไม่ได้
chǎn phûut phaa-sǎa Thai mâi dâai
I can’t speak Thai.
พูดไทยไม่เป็น
phûut Thai mâi bpen		 don’t know how to speak Thai 不會這項技能 เมื่อก่อนฉันพูดไทยไม่เป็น
mʉ̂a-gɔ̀ɔn chǎn phûut Thai mâi bpen
Before, I didn’t know how to speak Thai.

2. ต่อ vs เปลี่ยน｜接續 / 延長 vs 換掉

核心差異：ต่อ / dtɔ̀ɔ / continue; extend 是把原本的東西接下去；เปลี่ยน / bplìian / change; switch 是把 A 改成 B。
ThaiEnglish中文自然例句
ต่อวีซ่า
dtɔ̀ɔ wii-sâa		 extend / renew a visa續簽、延長已有簽證 ต้องไปต่อวีซ่าค่ะ
dtɔ̂ng bpai dtɔ̀ɔ wii-sâa khâ
I have to go extend my visa.
เปลี่ยนวีซ่า
bplìian wii-sâa		 change visa type換簽證類型 อยากเปลี่ยนวีซ่าจากท่องเที่ยวเป็นวีซ่านักเรียน
yàak bplìian wii-sâa jàak thɔ̂ɔng-thîiao bpen wii-sâa nák-riian
I want to change my visa from a tourist visa to a student visa.
ต่อเวลา
dtɔ̀ɔ wee-laa		 extend time延長時間 ขอต่อเวลาอีกสิบนาทีได้ไหมคะ
khɔ̌ɔ dtɔ̀ɔ wee-laa ìik sìp naa-thii dâai mái khá
Can I extend the time by ten more minutes?
เปลี่ยนเวลา
bplìian wee-laa		 change the time改時間 ขอเปลี่ยนเวลาเป็นพรุ่งนี้ได้ไหมคะ
khɔ̌ɔ bplìian wee-laa bpen phrûng-níi dâai mái khá
Can I change the time to tomorrow?

3. ครบ / ขาด / พอ / เสร็จ

ThaiRomanizationEnglish核心差異例句
ครบkhrópcomplete; all present該有的都有 เอกสารครบแล้วค่ะ
èek-gà-sǎan khróp lɛ́ɛw khâ
The documents are complete now.
ขาดkhàatmissing; lacking具體缺某一部分 เอกสารขาดสำเนาพาสปอร์ตค่ะ
èek-gà-sǎan khàat sǎm-nao pháat-bpɔ̀ɔt khâ
A passport copy is missing from the documents.
พอphɔɔenough數量 / 程度足夠 น้ำไม่พอค่ะ
náam mâi phɔɔ khâ

เสร็จsètfinished事情做完 โน้ตยังไม่เสร็จค่ะ
nóot yang mâi sèt khâ
The notes are not finished yet.

4. ไม่...นัก / ไม่ค่อย... / ไม่...เท่าไหร่

曼谷口語提醒：朋友聊天更常用 ไม่ค่อย... / mâi khɔ̂i... / not really... 或 ไม่...เท่าไหร่ / mâi ... thâo-rài / not really that...；ไม่...นัก 比較沉穩、文章感。
Pattern使用感例句
ไม่แพงนัก
mâi phɛɛng nák
not very expensive		 稍正式、理性 ร้านนี้ไม่แพงนัก
ráan níi mâi phɛɛng nák
This restaurant isn’t very expensive.
ไม่ค่อยแพง
mâi khɔ̂i phɛɛng
not really expensive		 最日常口語 ร้านนี้ไม่ค่อยแพง
ráan níi mâi khɔ̂i phɛɛng
This restaurant isn’t really expensive.
ไม่แพงเท่าไหร่
mâi phɛɛng thâo-rài
not that expensive		 口語柔和 คาเฟ่นี้ไม่แพงเท่าไหร่
gaa-fɛɛ níi mâi phɛɛng thâo-rài
This café isn’t that expensive.
Restaurant｜吃不完 / 分食 / 打包
จานนี้ใหญ่กว่าที่คิด กินไม่หมดแน่ ๆ
jaan níi yài gwàa thîi khít, gin mâi mòt nɛ̂ɛ-nɛ̂ɛ
This dish is bigger than I thought. I definitely can’t finish it.
ถ้ากินไม่หมด เดี๋ยวแบ่งให้เพื่อนก็ได้
thâa gin mâi mòt, dǐaw bɛ̀ng hâi phʉ̂an gɔ̂ɔ dâai
If I can’t finish it, I can share it with my friend.
รสนี้ไม่แรงนัก กินง่ายดี
rót níi mâi rɛɛng nák, gin ngâai dii
This flavor isn’t too strong. It’s easy to eat.
Errands｜忘東西 / 回去拿 / 來不及
เมื่อกี้ฉันลืมมือถือไว้ที่ร้าน
mʉ̂a-gîi chǎn lʉʉm mʉʉ-thʉ̌ʉ wái thîi ráan
I just left my phone at the shop.
งั้นเดี๋ยวแวะกลับไปเอาก่อนนะ
ngán dǐaw wɛ́ glàp bpai ao gɔ̀ɔn ná
Then I’ll stop by and go back to get it first.
ถ้ากลับไปเอาตอนนี้ อาจจะไปไม่ทัน
thâa glàp bpai ao dtɔɔn-níi, àat jà bpai mâi than
If I go back to get it now, I might not make it in time.
Visa Office｜文件 / 缺漏 / 有效期

กรุณากรอกส่วนนี้ให้ครบค่ะ
gà-rú-naa grɔ̀ɔk sùan níi hâi khróp khâ

pháat-bpɔ̀ɔt dtɔ̂ng mii aa-yú lʉ̌ea yàang nɔ́ɔi hòk dʉan
The passport must have at least six months of validity remaining.
BTS / Gym｜轉線 / 健身房 / 改時間
ต้องเปลี่ยนสายที่สยาม แล้วต่อรถไปอารีย์
dtɔ̂ng bplìian sǎai thîi Sà-yǎam, lɛ́ɛw dtɔ̀ɔ rót bpai Aa-rîi

ขอเปลี่ยนเป็นพรุ่งนี้ได้ไหมคะ วันนี้ไปไม่ทัน
khɔ̌ɔ bplìian bpen phrûng-níi dâai mái khá, wan-níi bpai mâi than
Can we change it to tomorrow? I can’t make it today.
วันนี้คนที่ยิมเยอะจนไม่อยากเล่นเวท
wan-níi khon thîi yim yə́ jon mâi yàak lên wêt
There are so many people at the gym today that I don’t want to do weight training.

Structures｜今日句型整理

1. V + ไม่ถูก｜不知道怎麼 V 才對

Pattern：Verb + ไม่ถูก / V + mâi thùuk / don’t know how to V properly

คำถามนี้ตอบไม่ถูกเลย
kham-thǎam níi dtɔ̀ɔp mâi thùuk ləəy
I really don’t know how to answer this question.

2. V + ไม่ออก｜V 不出來

Pattern：Verb + ไม่ออก / V + mâi ɔ̀ɔk / can’t get V out; can’t figure out

สองคำนี้คล้ายกันมาก แยกไม่ออก
sɔ̌ɔng kham níi khláai gan mâak, yɛ̂ɛk mâi ɔ̀ɔk
These two words are very similar. I can’t distinguish them.

3. Adj / State + จน + Result｜……到造成某結果

Pattern：Adj / State + จน + Result / ... jon ... / so...that...

เหนื่อยจนพูดไม่ออก
nʉ̀ueai jon phûut mâi ɔ̀ɔk
I’m so tired I can’t speak.

4. ลืม + Object + ไว้ที่ + Place｜把……忘在……

ลืมกุญแจไว้บนโต๊ะ
lʉʉm gun-jaae wái bon dtó
I left the keys on the table.

5. เปลี่ยนจาก A เป็น B｜從 A 改成 B

ขอเปลี่ยนจากวันจันทร์เป็นวันอังคารได้ไหมคะ
khɔ̌ɔ bplìian jàak wan jan bpen wan ang-khaan dâai mái khá
Can we change it from Monday to Tuesday?

6. ไม่ + Adj + นัก｜不算太……

คอนโดนี้ไม่ไกลนักจาก BTS
khaawn-doo níi mâi glai nák jàak BTS
This condo isn’t very far from the BTS.

Visa Office｜簽證與文件
ต้องเตรียมเอกสารอะไรบ้างคะ ถ้าจะต่อวีซ่า
dtɔ̂ng dtriiam èek-gà-sǎan à-rai bâang khá, thâa jà dtɔ̀ɔ wii-sâa
What documents do I need to prepare if I’m going to extend my visa?
เอกสารส่วนนี้ยังไม่ครบค่ะ
èek-gà-sǎan sùan níi yang mâi khróp khâ
This section of the documents is still incomplete.
วีซ่าของฉันใกล้หมดอายุแล้วค่ะ
wii-sâa khɔ̌ɔng chǎn glâi mòt aa-yú lɛ́ɛw khâ
My visa is about to expire.
พาสปอร์ตต้องมีอายุเหลืออย่างน้อยหกเดือน
pháat-bpɔ̀ɔt dtɔ̂ng mii aa-yú lʉ̌ea yàang nɔ́ɔi hòk dʉan
The passport must have at least six months of validity remaining.
BTS / MRT｜轉線、來不及、改行程
ถ้าออกช้ากว่านี้ ก็น่าจะไปไม่ทัน
thâa ɔ̀ɔk cháa gwàa níi, gɔ̂ɔ nâa-jà bpai mâi than
If we leave later than this, we probably won’t make it in time.
ต้องเปลี่ยนสายที่อโศกหรือสยามคะ
dtɔ̂ng bplìian sǎai thîi À-sòok rʉ̌ʉ Sà-yǎam khá
Do I need to change lines at Asok or Siam?
ถ้างั้นไม่แวะซื้อกาแฟแล้วกัน
thâa ngán mâi wɛ́ sʉ́ʉ gaa-fɛɛ lɛ́ɛw gan
In that case, let’s not stop by to buy coffee.
จากตรงนี้ต้องต่อรถอีกไหมคะ
jàak dtrong níi dtɔ̂ng dtɔ̀ɔ rót ìik mái khá
From here, do I need to transfer again?
Café / Restaurant｜點餐、吃不完、打包
เมนูเยอะจนเลือกไม่ถูกเลยค่ะ
mee-nuu yə́ jon lʉ̂ak mâi thùuk ləəy khâ
There are so many menu items that I really can’t choose.
จานนี้ใหญ่กว่าที่คิด กินไม่หมดแน่ ๆ
jaan níi yài gwàa thîi khít, gin mâi mòt nɛ̂ɛ-nɛ̂ɛ
This dish is bigger than I thought. I definitely can’t finish it.
ถ้ากินไม่หมด เดี๋ยวแบ่งให้เพื่อนก็ได้
thâa gin mâi mòt, dǐaw bɛ̀ng hâi phʉ̂an gɔ̂ɔ dâai
If I can’t finish it, I can share it with my friend.
ขอห่อกลับบ้านได้ไหมคะ
khɔ̌ɔ hɔ̀ɔ glàp bâan dâai mái khá
Can I have this packed to go?
Condo / Gym｜住處、健身房、忘東西
คอนโดนี้เหมาะกับคนที่ใช้ BTS บ่อย
khaawn-doo níi mɔ̀ gàp khon thîi chái BTS bɔ̀i
This condo is suitable for people who use the BTS often.
ห้องฟิตเนสที่คอนโดไม่ใหญ่ แต่คนไม่เยอะนัก
hɔ̂ɔng fít-nêt thîi khaawn-doo mâi yài, dtɛ̀ɛ khon mâi yə́ nák
The condo fitness room isn’t big, but it’s not very crowded.
เมื่อกี้ลืมที่ชาร์จไว้ที่ยิม
mʉ̂a-gîi lʉʉm thîi châat wái thîi yim
I just left my charger at the gym.
เดี๋ยวแวะกลับไปเอาก่อนนะ
dǐaw wɛ́ glàp bpai ao gɔ̀ɔn ná
I’ll stop by and go back to get it first.
Shopping / Mall｜選不出來、適不適合、語氣太強
สองสีนี้สวยทั้งคู่ เลือกไม่ถูกเลย
sɔ̌ɔng sǐi níi sǔai tháng khûu, lʉ̂ak mâi thùuk ləəy
Both colors are pretty. I really can’t choose.
อันนี้ก็น่าจะเหมาะกับเธอนะ
an níi gɔ̂ɔ nâa-jà mɔ̀ gàp thəə ná
This one should probably suit you.
คำนี้แรงไป ใช้กับพนักงานไม่เหมาะ
kham níi rɛɛng bpai, chái gàp phá-nák-ngaan mâi mɔ̀
This wording is too strong. It’s not suitable to use with staff.
ถ้างั้นเอาอันนี้ก็แล้วกันค่ะ
thâa ngán ao an níi gɔ̂ɔ lɛ́ɛw gan khâ
In that case, I’ll take this one.
Chat / Study｜聊天收尾、筆記還沒完整
วันนี้พอแค่นี้ก่อนนะ พรุ่งนี้ค่อยต่อ
wan-níi phɔɔ khɛ̂ɛ níi gɔ̀ɔn ná, phrûng-níi khɔ̂i dtɔ̀ɔ
Let’s stop here for today. We’ll continue tomorrow.
ยังไม่ครบค่ะ เดี๋ยวค่อยเก็บต่ออีกนิด
yang mâi khróp khâ, dǐaw khɔ̂i gèp dtɔ̀ɔ ìik nít
It’s not complete yet. I’ll continue collecting a bit more later.
พูดไม่ถูกเหมือนกัน แต่รู้สึกว่าอันนี้ไม่เหมาะนัก
phûut mâi thùuk mʉ̌ean gan, dtɛ̀ɛ rúu-sʉ̀k wâa an níi mâi mɔ̀ nák
I don’t know how to say it either, but I feel this isn’t very suitable.
คิดไม่ออกว่าจะตอบยังไงดี
khít mâi ɔ̀ɔk wâa jà dtɔ̀ɔp yang-ngai dii
I can’t think of how to reply.

ngán bɛ̀ng hâi phʉ̂an gɔ̂ɔ dâai, rʉ̌ʉ khɔ̌ɔ hɔ̀ɔ glàp bâan mái

Dialogue 2｜Visa Office：文件缺漏、簽證快到期

Officer
เอกสารยังไม่ครบค่ะ ขาดสำเนาพาสปอร์ต
èek-gà-sǎan yang mâi khróp khâ, khàat sǎm-nao pháat-bpɔ̀ɔt
The documents are still incomplete. A passport copy is missing.
Student
ต้องส่งเพิ่มวันนี้เลยไหมคะ

Dialogue 4｜Chat：假日去哪、想不出來

A
วันหยุดไปไหนมามั่ง
wan-yùt bpai nǎi maa mâang
Where did you go during the holiday?
B
ไม่ได้ไปไหนเลย อยู่บ้านทั้งวัน
mâi dâai bpai nǎi ləəy, yùu bâan tháng wan
I didn’t go anywhere. I stayed home all day.
A
ดีเหมือนกันนะ ฉันคิดไม่ออกว่าจะไปไหน
dii mʉ̌ean gan ná, chǎn khít mâi ɔ̀ɔk wâa jà bpai nǎi
That’s nice too. I couldn’t think of where to go.

1. 把所有「說不出來」都說成 พูดไม่ได้

Wrong
ฉันพูดไม่ได้ว่ารู้สึกยังไง
chǎn phûut mâi dâai wâa rúu-sʉ̀k yang-ngai
I can’t speak how I feel.
Better
ฉันพูดไม่ถูกว่ารู้สึกยังไง
chǎn phûut mâi thùuk wâa rúu-sʉ̀k yang-ngai
I don’t know how to say how I feel.
Why
พูดไม่ได้ / phûut mâi dâai / can’t speak 是能力或情況上不能說；พูดไม่ถูก / phûut mâi thùuk / don’t know how to say it properly 是不知道怎麼說才合適。

2. 把「忘在某地」說成只用 ลืม

Wrong
ลืมมือถือที่บ้าน
lʉʉm mʉʉ-thʉ̌ʉ thîi bâan
forgot phone at home
Better
ลืมมือถือไว้ที่บ้าน
lʉʉm mʉʉ-thʉ̌ʉ wái thîi bâan
I left my phone at home.
Why
ไว้ / wái / leave; keep 讓「東西被留在那裡」很清楚。

3. 把「改時間」說成 ต่อเวลา

Wrong
ขอต่อเวลาเป็นพรุ่งนี้ได้ไหมคะ
khɔ̌ɔ dtɔ̀ɔ wee-laa bpen phrûng-níi dâai mái khá
Can I extend the time to tomorrow?
Better
ขอเปลี่ยนเวลาเป็นพรุ่งนี้ได้ไหมคะ
khɔ̌ɔ bplìian wee-laa bpen phrûng-níi dâai mái khá
Can I change the time to tomorrow?
Why
ต่อเวลา / dtɔ̀ɔ wee-laa / extend time 是延長時間；เปลี่ยนเวลา / bplìian wee-laa / change the time 才是改時間。

4. 把 พูดแรง 理解成「說話大聲」

Wrong idea
เขาพูดแรง
khǎo phûut rɛɛng
He speaks harshly.
Better for “speak loudly”
เขาพูดเสียงดัง
khǎo phûut sǐang dang
He speaks loudly.
Why
พูดแรง / phûut rɛɛng / speak harshly 是內容或語氣很重、很衝；不是單純音量大。

Practice｜今日練習

A. Vocabulary Recall｜看中文想泰文

  1. 吃不完
  2. 把手機忘在家
  3. 續簽 / 延長簽證
  4. 資料缺了一部分
  5. 想不出來要吃什麼
  6. 可以改成明天嗎？
Answer
  1. กินไม่หมด / gin mâi mòt / can’t finish eating it
  2. ลืมมือถือไว้ที่บ้าน / lʉʉm mʉʉ-thʉ̌ʉ wái thîi bâan / left my phone at home
  3. ต่อวีซ่า / dtɔ̀ɔ wii-sâa / extend / renew a visa
  4. ข้อมูลขาดบางส่วน / khɔ̂ɔ-muun khàat baang sùan / some information is missing
  5. คิดไม่ออกว่าจะกินอะไร / khít mâi ɔ̀ɔk wâa jà gin à-rai / can’t think of what to eat
  6. เปลี่ยนเป็นพรุ่งนี้ได้ไหมคะ / bplìian bpen phrûng-níi dâai mái khá / Can we change it to tomorrow?

B. Choose the right word｜選詞

  1. 我不知道怎麼回答才好：ตอบไม่___（ถูก / ออก）
  2. 我想不出來：คิดไม่___（ถูก / ออก）
  3. 簽證快到期：วีซ่าใกล้___（ครบ / หมดอายุ）
  4. 改時間：___เวลา（ต่อ / เปลี่ยน）
Answer
  1. ตอบไม่ถูก / dtɔ̀ɔp mâi thùuk / don’t know how to answer properly
  2. คิดไม่ออก / khít mâi ɔ̀ɔk / can’t think of it
  3. วีซ่าใกล้หมดอายุ / wii-sâa glâi mòt aa-yú / visa is about to expire
  4. เปลี่ยนเวลา / bplìian wee-laa / change the time
