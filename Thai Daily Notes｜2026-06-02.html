4. 多義詞不要背散：ถูก / ลง / นับ

今天的多義詞重點是「核心感」：不要只背第一個英文意思。

ThaiRomanizationEnglish真正要記的語感

5. 生活情緒與禮貌距離：不是字典意思，而是怎麼講才自然

เป็นอะไร bpen à-rai What’s wrong?
不是「是什麼」，而是「怎麼了 / 還好嗎」。

สะดวกไหม sà-dùak mái Is it convenient? / Are you available?
比直接問「你有空嗎」更自然、有禮貌。

อาย aai embarrassed vs ขี้อาย khîi-aai shy
อาย 是當下不好意思；ขี้อาย 是個性容易害羞。

6. 今日最值得背的一句 + Pattern

Daily Sentence

อันนี้ไม่นับนะ
an-níi mâi náp ná
This doesn’t count.

生活語感像中文：「這個不算喔 / 這次不算啦」。比單背 นับเลข 更實用。

Daily Pattern

ไม่รู้ก็เลยถาม
mâi rúu gɔ̂ɔ ləəy thǎam
I didn’t know, so I asked.

ก็เลย gɔ̂ɔ ləəy 是很生活的「所以就……」，比書面連接詞更像聊天。

Usage Notes｜重點比較與曼谷語感

การ vs ความ vs กัน｜三個 gaan / khwaam / gan 不要混

ThaiRomanizationEnglish中文語感例句
การgaanthe act of; -ing把動作變名詞：……這件事การเรียนภาษาไทยไม่ง่าย

ความkhwaamabstract noun marker把狀態、性質變抽象名詞ความมั่นใจของเธอดีขึ้นมาก

一句話記：การ 是「動作這件事」；ความ 是「抽象狀態」；กัน 是「一起 / 互相」。

Structures｜今日句型

Pattern 1｜แค่...เท่านั้น = 只是……而已

ฉันแค่อยากช่วยเท่านั้น
chǎn khɛ̂ɛ yàak chûai thâo-nán
I only wanted to help.

นี่เป็นแค่ตัวอย่างเท่านั้น
nîi bpen khɛ̂ɛ dtua-yàang thâo-nán
This is only an example.

Pattern 2｜A ก็เลย B = A，所以就 B

ไม่รู้ก็เลยถาม
mâi rúu gɔ̂ɔ ləəy thǎam
I didn’t know, so I asked.

ตื่นสายก็เลยไม่ทัน
dtʉ̀ʉn sǎai gɔ̂ɔ ləəy mâi than
I woke up late, so I wasn’t in time.

Pattern 3｜นับว่า = 算是

แบบนี้ก็นับว่าดีแล้ว
bɛ̀ɛp níi gɔ̂ɔ náp wâa dii lɛ́ɛw
This can already be considered good.

อันนี้ไม่นับนะ
an-níi mâi náp ná
This doesn’t count.

Pattern 4｜Adj + ขึ้น = 變得更……

ภาษาไทยของเธอดีขึ้นมาก
phaa-sǎa thai khɔ̌ɔng thəə dii khʉ̂n mâak
Your Thai has improved a lot.

ตอนนี้อ่านง่ายขึ้นแล้ว
dtɔɔn-níi àan ngâai khʉ̂n lɛ́ɛw
Reading is easier now.

Pattern 5｜ได้...จาก... = 從……得到……

ฉันได้ความรู้จากหนังสือเล่มนี้
chǎn dâai khwaam-rúu jàak nǎng-sʉ̌ʉ lêm níi
I got knowledge from this book.

เธอได้คำนี้มาจากไหน
thəə dâai kham níi maa jàak nǎi
Where did you learn this word from?

Pattern 6｜ขี้ + Adj / Verb = 容易……的人

ฉันขี้ลืมมากเลย
chǎn khîi-lʉʉm mâak ləəy
I’m really forgetful.

ฉันขี้ง่วงตอนบ่าย
chǎn khîi-ngûang dtɔɔn bàai
I get sleepy easily in the afternoon.

คุณสะดวกประชุมบ่ายสองไหม

Friends｜邊吃邊聊、想來想去
เรากินไปคุยไป สนุกดี
rao gin bpai khui bpai, sà-nùk dii
We ate while chatting. It was fun.
คิดไปคิดมาก็ยังไม่รู้จะทำยังไง
khít bpai khít maa gɔ̂ɔ yang mâi rúu jà tham yang-ngai
After thinking it over, I still don’t know what to do.
คุยไปคุยมาก็เป็นเพื่อนกัน
khui bpai khui maa gɔ̂ɔ bpen phʉ̂an gan
We talked and talked and eventually became friends.
Emotion｜只是、害羞、怎麼了
ฉันแค่ถามเท่านั้น ไม่ได้โกรธ
chǎn khɛ̂ɛ thǎam thâo-nán, mâi dâai gròot
I was only asking. I wasn’t angry.
เป็นไรอะ ทำหน้าเครียดจัง
bpen-rai à, tham nâa khrîiat jang
What’s wrong? You look stressed.
เขินนิดหน่อย แต่ไม่ได้โกรธ
khə̌ən nít nɔ̀i, dtɛ̀ɛ mâi dâai gròot
I’m a little bashful, but I’m not angry.

Mistakes｜常見錯誤修正

1. 把 ง่วง 和 เหนื่อย 混在一起

Wrong
ฉันเหนื่อย อยากนอน
chǎn nʉ̀ai, yàak nɔɔn
I’m tired and want to sleep.
Better
ฉันง่วง อยากนอน
chǎn ngûang, yàak nɔɔn
I’m sleepy and want to sleep.
Why：เหนื่อย 是累；ง่วง 是想睡。

2. 把 สะดวก 只理解成 convenient

Too literal
สะดวกคุยไหม
sà-dùak khui mái
Is talking convenient?
Natural meaning
สะดวกคุยไหม
sà-dùak khui mái
Are you available to talk?
Why：這裡是在問「現在方便 / 有空聊嗎」。

3. 把 ถูก 全部當便宜

Wrong mindset
ถูกถาม
thùuk thǎam
cheap ask
Better
ถูกถาม
thùuk thǎam
be asked
Why：ถูก + V 是被動，不是便宜。

4. ฉันไปด้วยกัน 不如 ฉันไปด้วย 自然

Less natural
ฉันไปด้วยกัน
chǎn bpai dûai gan
I go together.
Better
ฉันไปด้วย
chǎn bpai dûai
I’ll go too.
Why：主詞是「我」時，通常用 ไปด้วย；ไปด้วยกัน 強調「我們一起去」。

5. 把 การ / ความ / กัน 混在一起

Wrong
กันเรียนภาษาไทยไม่ง่าย
gan-riian phaa-sǎa thai mâi ngâai
Together studying Thai is not easy.
Better
การเรียนภาษาไทยไม่ง่าย
gaan-riian phaa-sǎa thai mâi ngâai
Studying Thai is not easy.
Why：การ 名詞化動作；กัน 是一起。
