Comparison Sets｜今日必複習比較組

  • โดย / ผ่าน / ทาง / ด้วย / ใช้ / จาก：英文 by、中文「透過」在泰文裡要依情境換字。
  • เมื่อ / ตอนที่ / เวลา / สมัยที่ / พอ：都是 when 類，但語感不同。
  • บาง / บ้าง：一個放名詞前，一個常放句尾或動詞後。
  • เยอะ / มาก / หลาย：數量多、程度很、好幾個。
  • ก็ได้ / ก็แล้วกัน / แล้วกัน：接受、收尾決定、短版收尾。
  • เกรงใจ / กลัว / อาย / รบกวน：不好意思麻煩、害怕、害羞尷尬、打擾。
  • ถูกใจ / ชอบ / พอใจ / ประทับใจ：中意、喜歡、滿意、印象深刻。

ตลอดชีวิต dtà-lòt chii-wít throughout one’s life; for one’s whole life Practical Usage 語氣有重量，常用於人生經驗、承諾、深刻記憶。日常也常用 ทั้งชีวิต / tháng chii-wít / one’s whole life。 ฉันจะไม่ลืมเรื่องนี้ตลอดชีวิต
chǎn jà mâi lʉʉm rʉ̂ang níi dtà-lòt chii-wít
I will never forget this for the rest of my life.
โดย dooi by; via; by means of Textbook-based 正式、書面、被動句、交通方式說明常用。日常交通更自然說 นั่งเครื่องบินมา / nâng khrʉ̂ang-bin maa / came by plane。 หนังสือเล่มนี้เขียนโดยนักเขียนไทย
nǎng-sʉ̌ue lêm níi khǐian dooi nák-khǐian Thai
This book was written by a Thai writer.
ผ่าน phàan through; via Local usage 平台、朋友、管道最自然的「透過」。比 โดย 更口語生活。 เราคุยกันผ่านไลน์ได้ค่ะ
rao khui gan phàan Lai dâai khâ
We can talk via Line.
ต่าง dtàang different; separate; various Practical Usage 日常常用 ต่างกัน / dtàang gan / different from each other。固定詞 ต่างประเทศ 是「國外」。 สองคำนี้ต่างกันยังไงคะ
sǎawng kham níi dtàang gan yang-ngai khá
How are these two words different?
เดี๋ยวก่อน dǐiao gɔ̀ɔn hold on; wait a moment Local usage 叫對方或自己「先等一下、先不要急」。不是「晚點再做」。 เดี๋ยวก่อนนะคะ ขอคิดแป๊บนึง
dǐiao gɔ̀ɔn ná khá, khɔ̌ɔ khít bpáep nueng
One moment, please. Let me think for a second.
เดี๋ยวค่อย... dǐiao khɔ̂i... later then...; do ... later Local usage 某件事晚點再做。聊天中 เดี๋ยวค่อยคุย 可能偏冷，加 กันนะ 比較柔和。 เดี๋ยวว่างแล้วค่อยคุยกันนะ
dǐiao wâang láaeo khɔ̂i khui gan ná
When I’m free, let’s talk later, okay?
เพิ่ง phə̂ng just; just recently Practical Usage 放在動詞前，表示剛剛才、最近才。固定搭配 อย่าเพิ่ง / yàa phə̂ng / don’t ... yet。 เพิ่งเห็นข้อความค่ะ
phə̂ng hěn khɔ̂ɔ-khwaam khâ
I just saw the message.
ตอนที่ dtɔɔn thîi when; at the time that Local usage 日常最安全的「當……的時候」，適合具體事件、回憶、背景時間。 ตอนที่ฉันอยู่กรุงเทพฯ ฉันชอบไปคาเฟ่
dtɔɔn thîi chǎn yùu grung-thêep, chǎn châawp bpai khaa-fêe
When I was in Bangkok, I liked going to cafés.
พอ...ก็... phɔɔ...gâw... once..., then...; as soon as... Local usage 強調 A 發生後 B 接著發生。不是單純背景時間，而是事件觸發。 พอถึงบ้านก็หลับเลย
phɔɔ thʉ̌eng bâan gâw làp ləəy
Once I got home, I fell asleep right away.
ตาม dtaam follow; according to; along Practical Usage 核心是「跟著某個人、路線、規則、計畫走」。 ไปก่อนเลย เดี๋ยวตามไป
bpai gɔ̀ɔn ləəy, dǐiao dtaam bpai
Go ahead first. I’ll follow later.
เกรงใจ greeng-jai feel bad to impose; be considerate Local usage 泰國文化核心詞：不好意思麻煩、怕造成別人負擔。不是單純 shy。 ฉันเกรงใจคุณมากค่ะ
chǎn greeng-jai khun mâak khâ
I feel really bad about troubling you.
ถูกใจ thùuk-jai be to one’s taste; be pleased with Local usage 比 ชอบ / châawp / like 更有「打中喜好、很中意」的感覺。 อันนี้ถูกใจฉันมาก
an níi thùuk-jai chǎn mâak
This one is really to my taste.
ต่อ dtɔ̀ɔ continue; connect; transfer; per Practical Usage 核心是「接續、連接」。可表示繼續、轉車、排隊、殺價、每單位。 วันนี้พอแค่นี้ก่อน พรุ่งนี้ค่อยต่อ
wan-níi phɔɔ khâe níi gɔ̀ɔn, phrûng-níi khɔ̂i dtɔ̀ɔ
That’s enough for today. We’ll continue tomorrow.
แบบ bàep type; style; model; pattern; like Local usage 樣子、類型、款式、方式；口語中也像 English “like / kind of”。 ช่วยอธิบายแบบง่าย ๆ ได้ไหมคะ
chûai à-thí-baai bàep ngâai-ngâai dâai mái khá
Can you explain it in a simple way?
บาง baang some; certain; partial Practical Usage 放在名詞前，表示「某些、有些」。注意跟 บ้าง / bâang / some; a bit 不同。 บางอย่างอธิบายยาก
baang yàang à-thí-baai yâak
Some things are hard to explain.
บ้าง bâang some; a bit; among others Common mistake 常放句尾或動詞後，表示一些、多少、有哪些。聲調是 falling tone。 พูดไทยได้บ้างค่ะ
phûut Thai dâai bâang khâ
I can speak some Thai.
ตรงกลาง dtrong glaang in the middle; at the center Practical Usage 具體位置的正中間。若是 A 和 B 之間，用 ระหว่าง / rá-wàang / between。 วางไว้ตรงกลางโต๊ะได้เลยค่ะ
waang wái dtrong glaang dtó dâai ləəy khâ
You can put it in the middle of the table.
งั้น ngán then; in that case Local usage 最口語的「那……、那就……」。比 ถ้าอย่างนั้น 短很多。 งั้นพักก่อนนะ ไว้ค่อยคุยกัน
ngán phák gɔ̀ɔn ná, wái khɔ̂i khui gan
Then rest first. We can talk later.
เยอะ yóe a lot; many; much Local usage 口語高頻「很多」。偏數量多，不是所有「很」都能用。คิดเยอะ 是「想太多」。 วันนี้คนเยอะมาก
wan-níi khon yóe mâak
There are so many people today.
ก็แล้วกัน gâw láaeo gan then; let’s settle on that; I guess... then Local usage 做決定、選東西、收尾、妥協時用。語氣可能自然，也可能有「算了」感。 งั้นเอาอันนี้ก็แล้วกันค่ะ
ngán ao an níi gâw láaeo gan khâ
Then I’ll take this one.

1. โดย / ผ่าน / ทาง / ด้วย / ใช้ / จาก｜by / through / via 怎麼自然說？

核心差異：中文「透過」和英文 by / through / via 不要全部翻成 โดย / dooi / by。泰文會依「交通、平台、工具、來源、正式管道」換字。
情境自然泰文語感例句
正式 by / 被動 โดย
dooi
by; via		 書面、正式、課本清楚 หนังสือเล่มนี้เขียนโดยนักเขียนไทย
nǎng-sʉ̌ue lêm níi khǐian dooi nák-khǐian Thai
This book was written by a Thai writer.
平台 / 朋友 / 管道 ผ่าน
phàan
through; via		 生活中最自然的「透過」 เราเจอกันผ่านเพื่อนค่ะ
rao jəə gan phàan phʉ̂an khâ
We met through a friend.
正式聯絡管道 ทาง
thaang
via; by way of		 客服、文件、商務管道 ส่งเอกสารทางอีเมลได้ไหมคะ
sòng èek-gà-sǎan thaang ii-meeo dâai mái khá
Can I send the documents by email?
工具 / 付款方式 ด้วย
dûai
with; by means of		 用某工具完成 จ่ายด้วยบัตรเครดิตได้ไหมคะ
jàai dûai bàt khree-dìt dâai mái khá
Can I pay with a credit card?
使用某工具做事 ใช้
chái
use		 強調「使用」 ฉันใช้ ChatGPT เรียนภาษาไทย
chǎn chái ChatGPT riian phaa-sǎa Thai
I use ChatGPT to learn Thai.
從某人 / 經驗得知 จาก
jàak
from		 來源、起點 ฉันรู้เรื่องนี้จากเพื่อน
chǎn rúu rʉ̂ang níi jàak phʉ̂an
I heard about this from a friend.

2. เมื่อ / ตอนที่ / เวลา / สมัยที่ / พอ｜五種 when

快速判斷：日常具體事件用 ตอนที่；一般習慣用 เวลา；正式書面用 เมื่อ；以前某時期用 สมัยที่；一……就……用 พอ...ก็...。
ThaiRomanizationEnglish用法例句
เมื่อ mʉ̂a when; at the time when 正式、書面、公告、故事旁白 เมื่อได้รับเอกสารแล้ว กรุณาติดต่อเจ้าหน้าที่
mʉ̂a dâai-ráp èek-gà-sǎan láaeo, gà-rú-naa dtìt-dtɔ̀ɔ jâo-nâa-thîi
When you have received the documents, please contact the staff.
ตอนที่ dtɔɔn thîi when; at the time that 日常最安全的「當……的時候」 ตอนที่มาถึงแล้ว บอกฉันหน่อยนะ
dtɔɔn thîi maa thʉ̌eng láaeo, bɔ̀ɔk chǎn nɔ̀i ná
When you arrive, tell me, okay?
เวลา wee-laa when; whenever 每當、一般情況、習慣 เวลาคนไทยพูดเร็ว ฉันฟังไม่ทัน
wee-laa khon Thai phûut reo, chǎn fang mâi than
When Thai people speak fast, I can’t keep up.
สมัยที่ sà-mǎi thîi back when; in the period when 以前某個時期、學生時代、小時候 สมัยที่เรียนมหาลัย ฉันทำงานพาร์ตไทม์ด้วย
sà-mǎi thîi riian má-hǎa-lai, chǎn tham-ngaan pháat-thai dûai
Back when I was in university, I also worked part-time.
พอ...ก็... phɔɔ...gâw... once..., then... 一……就……，強調事件接續 พอเข้าร้าน พนักงานก็ถามว่าเอาอะไรดีคะ
phɔɔ khâo ráan, phá-nák-ngaan gâw thǎam wâa ao à-rai dii khá
As soon as I entered the shop, the staff asked, “What would you like?”

3. เกรงใจ / กลัว / อาย / รบกวน

核心差異：เกรงใจ 是怕麻煩別人；กลัว 是害怕；อาย 是害羞或尷尬；รบกวน 是打擾、麻煩到對方。

ThaiMeaningExample
เกรงใจ
greeng-jai
feel bad to impose		 不好意思麻煩、顧慮對方 ฉันเกรงใจ ไม่อยากรบกวนเขา
chǎn greeng-jai, mâi yàak róp-guan khǎo
I feel bad; I don’t want to bother him/her.
กลัว
glua
afraid; scared		 真的害怕 ฉันกลัวหมา
chǎn glua mǎa
I’m afraid of dogs.
อาย
aai
embarrassed; shy		 尷尬、害羞、丟臉 ฉันอายเวลาพูดภาษาไทยผิด
chǎn aai wee-laa phûut phaa-sǎa Thai phìt
I feel embarrassed when I speak Thai incorrectly.
รบกวน
róp-guan
bother; trouble		 打擾、麻煩別人 ขอโทษที่รบกวนนะคะ
khɔ̌ɔ-thôot thîi róp-guan ná khá
Sorry for bothering you.

4. ถูกใจ / ชอบ / พอใจ / ประทับใจ

核心差異：ชอบ 是普通喜歡；ถูกใจ 是打中喜好；พอใจ 是滿意結果；ประทับใจ 是印象深刻、被打動。

ThaiMeaningExample
ชอบ
châawp
like		 一般喜歡 ฉันชอบร้านนี้
chǎn châawp ráan níi
I like this shop.
ถูกใจ
thùuk-jai
be to one’s taste		 合心意、很中意 ร้านนี้ถูกใจฉันมาก
ráan níi thùuk-jai chǎn mâak
This shop is really to my taste.
พอใจ
phɔɔ-jai
satisfied		 對結果、標準滿意 ฉันพอใจกับผลลัพธ์นี้
chǎn phɔɔ-jai gàp phǒn-láp níi
I’m satisfied with this result.
ประทับใจ
bprà-tháp-jai
impressed		 印象深刻、被打動 ฉันประทับใจบริการของร้านนี้มาก
chǎn bprà-tháp-jai baw-rí-gaan khɔ̌ɔng ráan níi mâak
I’m very impressed by this restaurant’s service.

5. บาง / บ้าง

Common mistake：บาง / baang / some; certain 放名詞前；บ้าง / bâang / some; a bit 常放句尾或動詞後。
Thai位置Example
บาง
baang
some; certain		 名詞前 บางคนพูดไทยเร็วมาก
baang khon phûut Thai reo mâak
Some people speak Thai very fast.
บ้าง
bâang
some; a bit; among others		 句尾 / 動詞後 คุณชอบกินอะไรบ้าง
khun châawp gin à-rai bâang
What kinds of things do you like to eat?

6. เยอะ / มาก / หลาย

Thai核心語感Example
เยอะ
yóe
a lot		 數量很多、口語 ข้อความเยอะมาก อ่านไม่ทัน
khɔ̂ɔ-khwaam yóe mâak, àan mâi than

มาก
mâak
very; much		 程度很，最通用 วันนี้ฉันเหนื่อยมาก
wan-níi chǎn nʉ̀ai mâak

หลาย
lǎai
many; several		 好幾個、好幾種，常接量詞 / 名詞 มีหลายแบบให้เลือก
mii lǎai bàep hâi lʉ̂ak
There are many styles to choose from.

7. ก็ได้ / ก็แล้วกัน / แล้วกัน

Thai核心語感Example
ก็ได้
gâw dâai
okay; fine		 接受、也可以，有時是妥協 ร้านนี้ก็ได้
ráan níi gâw dâai

ก็แล้วกัน
gâw láaeo gan
let’s settle on that		 做最後決定、收尾 ร้านนี้ก็แล้วกัน
ráan níi gâw láaeo gan

แล้วกัน
láaeo gan
then; settle on that		 更短、更口語 เอาอันนี้แล้วกัน
ao an níi láaeo gan
I’ll take this one then.

8. แบบ / เหมือน

แบบ / bàep / type; style; like 強調「方式、樣子、類型」；เหมือน / mʉ̌an / same as; like 強調「相似」。

เขาพูดเหมือนครู
khǎo phûut mʉ̌an khruu
He/She speaks like a teacher.

เขาพูดแบบครู
khǎo phûut bàep khruu
He/She speaks in a teacher-like way.

Structures｜今日句型整理

1. เดี๋ยวค่อย + Verb

意思：等一下再……、晚點再……。適合把動作延後。

เดี๋ยวค่อยดูนะ
dǐiao khɔ̂i duu ná
I’ll look later, okay?

เดี๋ยวค่อยบอกนะ
dǐiao khɔ̂i bɔ̀ɔk ná
I’ll tell you later, okay?

2. เพิ่ง + Verb

意思：剛剛才、最近才。เพิ่ง 放在動詞前。

ฉันเพิ่งถึงบ้านค่ะ
chǎn phə̂ng thʉ̌eng bâan khâ
I just got home.

ฉันเพิ่งเคยกินอาหารเหนือ
chǎn phə̂ng khəəi gin aa-hǎan nʉ̌a
I just tried Northern Thai food for the first time.

3. อย่าเพิ่ง + Verb

意思：先不要……、還不要急著……。

อย่าเพิ่งไปนะ
yàa phə̂ng bpai ná
Don’t go yet.

อย่าเพิ่งรีบตัดสินใจ
yàa phə̂ng rîip dtàt-sǐn-jai
Don’t rush to decide yet.

4. พอ + A + ก็ + B

意思：一 A 就 B。強調事件接續或反應。

พอเห็นข้อความก็รีบตอบ
phɔɔ hěn khɔ̂ɔ-khwaam gâw rîip dtɔ̀ɔp
As soon as I saw the message, I replied quickly.

5. ตามที่ + clause

意思：依照……所說 / 所約定 / 所理解。

ตามที่คุณบอก ฉันจะส่งเอกสารให้วันนี้
dtaam thîi khun bɔ̀ɔk, chǎn jà sòng èek-gà-sǎan hâi wan-níi
As you said, I’ll send the documents today.

ตามที่ตกลงกันไว้ เราจะเริ่มประชุมตอนสิบโมง
dtaam thîi dtòk-long gan wái, rao jà rə̂əm bprà-chum dtɔɔn sìp moong
As agreed, we will start the meeting at 10.

6. Verb + ต่อ

意思：繼續做某事。

พรุ่งนี้ค่อยทำต่อนะ
phrûng-níi khɔ̂i tham dtɔ̀ɔ ná
Let’s continue doing it tomorrow.

คืนนี้ดูซีรีส์ต่อไหม
khʉʉn-níi duu sii-rîi dtɔ̀ɔ mái
Do you want to continue watching the series tonight?

7. งั้น + decision + ก็แล้วกัน

意思：那就……好了；做收尾決定。

งั้นเอาแบบนี้ก็แล้วกัน
ngán ao bàep níi gâw láaeo gan
Then let’s do it this way.

งั้นไว้พรุ่งนี้ก็แล้วกัน
ngán wái phrûng-níi gâw láaeo gan
Then let’s leave it for tomorrow.

8. ต่อให้...ก็...

意思：就算……也……。

ต่อให้ยาก ฉันก็อยากลอง
dtɔ̀ɔ hâi yâak, chǎn gâw yàak laawng
Even if it’s difficult, I still want to try.

oo-khee, ngán dəən dtaam Google Maps bpai gɔ̀ɔn ná
Okay, then let’s follow Google Maps first.

Dialogue 4｜Friends / Chat

A
วันนี้ฉันงานเยอะมาก อาจจะไปไม่ได้
wan-níi chǎn ngaan yóe mâak, àat-jà bpai mâi dâai
I have a lot of work today. I may not be able to go.
B
ถ้างั้นไว้วันหลังก็ได้ ไม่ต้องเกรงใจนะ
thâa ngán wái wan lǎng gâw dâai, mâi dtɔ̂ng greeng-jai ná
Then another day is fine. Don’t feel bad.
A
ขอบคุณนะ เกรงใจมากเลย
khɔ̀ɔp-khun ná, greeng-jai mâak ləəy
Thank you. I really feel bad.
Wrong
คุยโดยไลน์
khui dooi Lai
talk by Line
Better
คุยผ่านไลน์
khui phàan Lai
talk via Line
Why
โดย / dooi / by 偏正式；平台、管道日常更自然用 ผ่าน / phàan / through; via。
Wrong
ฉันใช้เครื่องบินมา
chǎn chái khrʉ̂ang-bin maa
I used an airplane to come.
Better
ฉันนั่งเครื่องบินมา
chǎn nâng khrʉ̂ang-bin maa
I came by plane.
Why
交通工具日常用 นั่ง + vehicle + มา/ไป / nâng + vehicle + maa/bpai / come/go by vehicle。
Wrong
เดี๋ยวก่อนกิน
dǐiao gɔ̀ɔn gin
wait first eat
Better
เดี๋ยวค่อยกิน
dǐiao khɔ̂i gin
I’ll eat later.
Why
เดี๋ยวก่อน 是叫人先等；「等一下再吃」要用 เดี๋ยวค่อยกิน。
Wrong
ฉันกินเพิ่ง
chǎn gin phə̂ng
I ate just
Better
ฉันเพิ่งกิน
chǎn phə̂ng gin
I just ate.
Why
เพิ่ง / phə̂ng / just recently 通常放在動詞前。
Wrong
บ้างคน
bâang khon
some people
Better
บางคน
baang khon
some people
Why
บาง 放名詞前；บ้าง 常放句尾或動詞後。
Wrong
พูดไทยได้บาง
phûut Thai dâai baang
can speak some Thai
Better
พูดไทยได้บ้าง
phûut Thai dâai bâang
can speak some Thai
Why
「會一些」用句尾 บ้าง / bâang / some; a bit。
Wrong
ฉันเหนื่อยเยอะ
chǎn nʉ̀ai yóe
I’m tired a lot.
Better
ฉันเหนื่อยมาก
chǎn nʉ̀ai mâak
I’m very tired.
Why
เยอะ 是數量多；情緒或狀態程度用 มาก。
Wrong
ฉันเกรงใจหมา
chǎn greeng-jai mǎa
I feel bad imposing on dogs.
Better
ฉันกลัวหมา
chǎn glua mǎa
I’m afraid of dogs.
Why
เกรงใจ 是怕麻煩別人；害怕用 กลัว。
Wrong
งั้นก็แล้วกัน
ngán gâw láaeo gan
Then so be it.
Better
งั้นไว้ค่อยคุยกันนะ
ngán wái khɔ̂i khui gan ná
Then let’s talk later, okay?
Why
單獨說 งั้นก็แล้วกัน 可能偏冷，像「那就這樣吧」。想柔和收尾要補具體動作。
Wrong
สมัยที่ฉันกินข้าว โทรศัพท์ดัง
sà-mǎi thîi chǎn gin khâao, thoo-rá-sàp dang
Back when I was eating, the phone rang.
Better
ตอนที่ฉันกินข้าว โทรศัพท์ดัง
dtɔɔn thîi chǎn gin khâao, thoo-rá-sàp dang
When I was eating, the phone rang.
Why
สมัยที่ 用於一段時期，不適合短暫動作。

Practice A｜Vocabulary Recall

  1. 「剛看到訊息」泰文怎麼說？
  2. 「人很多 / 很擠」泰文怎麼說？
  3. 「我會說一點泰文」泰文怎麼說？
  4. 「那就這個好了」泰文怎麼說？
  5. 「不用不好意思」泰文怎麼說？
  6. 「這兩個詞差在哪？」泰文怎麼說？
Answer
  1. เพิ่งเห็นข้อความ / phə̂ng hěn khɔ̂ɔ-khwaam / just saw the message
  2. คนเยอะ / khon yóe / many people; crowded
  3. พูดไทยได้บ้างค่ะ / phûut Thai dâai bâang khâ / I can speak some Thai.
  4. เอาอันนี้ก็แล้วกัน / ao an níi gâw láaeo gan / I’ll take this one then.
  5. ไม่ต้องเกรงใจ / mâi dtɔ̂ng greeng-jai / No need to feel bad.
  6. สองคำนี้ต่างกันยังไงคะ / sǎawng kham níi dtàang gan yang-ngai khá / How are these two words different?

Practice B｜Choose the right word

  1. 透過 Line 聊：โดย / ผ่าน
  2. 有些人：บางคน / บ้างคน
  3. 很累：เหนื่อยมาก / เหนื่อยเยอะ
  4. 晚點再聊：เดี๋ยวค่อยคุย / เดี๋ยวก่อนคุย
  5. 一到家就睡：ตอนที่ถึงบ้านก็หลับ / พอถึงบ้านก็หลับ
  6. 不好意思麻煩你：อาย / เกรงใจ
Answer
  1. ผ่าน → คุยผ่านไลน์ / khui phàan Lai / talk via Line
  2. บางคน / baang khon / some people
  3. เหนื่อยมาก / nʉ̀ai mâak / very tired
  4. เดี๋ยวค่อยคุย / dǐiao khɔ̂i khui / talk later
  5. พอถึงบ้านก็หลับ / phɔɔ thʉ̌eng bâan gâw làp / once got home, fell asleep
  6. เกรงใจ / greeng-jai / feel bad to impose

Practice C｜Chinese → Thai

  1. 那我先走喔。
  2. 可以用簡單一點的方式解釋嗎？
  3. 你先去，我等一下過去。
  4. 那就改天好了。
  5. 我真的很不好意思麻煩你。
  6. 這間房我很中意。
Answer
  1. งั้นฉันไปก่อนนะ
    ngán chǎn bpai gɔ̀ɔn ná
    Then I’ll go first.
  2. ช่วยอธิบายแบบง่าย ๆ ได้ไหมคะ
    chûai à-thí-baai bàep ngâai-ngâai dâai mái khá
    Can you explain it in a simple way?
  3. ไปก่อนเลย เดี๋ยวตามไป
    bpai gɔ̀ɔn ləəy, dǐiao dtaam bpai
    Go ahead first. I’ll follow later.
  4. ไว้วันหลังก็แล้วกัน
    wái wan lǎng gâw láaeo gan
    Another day then.
  5. ฉันเกรงใจคุณมากค่ะ
    chǎn greeng-jai khun mâak khâ
    I feel really bad about troubling you.
  6. ห้องนี้ถูกใจฉันมาก
    hâwng níi thùuk-jai chǎn mâak
    This room is really to my taste.

Practice D｜Mini Dialogue Completion

  1. A: ร้านนี้คนเยอะมาก。B: ________（那要不要去別家？）
  2. A: วันนี้ไม่ว่างอะ。B: ________（那就改天也可以。）
  3. Staff: ชาไทยหมดค่ะ。You: ________（那就冰咖啡好了。）
  4. A: วันนี้เหนื่อยมาก。B: ________（那你先休息喔，晚點再聊。）
  5. Teacher: วันนี้พอแค่นี้ก่อน。Student: ________（明天再繼續。）
Answer
  1. งั้นไปร้านอื่นไหม
    ngán bpai ráan ùuen mái
    Then should we go to another place?
  2. ถ้างั้นไว้วันหลังก็ได้
    thâa ngán wái wan lǎng gâw dâai
    Then another day is fine.
  3. งั้นเอากาแฟเย็นก็แล้วกันค่ะ
    ngán ao gaa-fae yen gâw láaeo gan khâ
    Then I’ll take iced coffee.
  4. งั้นพักก่อนนะ ไว้ค่อยคุยกัน
    ngán phák gɔ̀ɔn ná, wái khɔ̂i khui gan
    Then rest first. We can talk later.
  5. พรุ่งนี้ค่อยต่อค่ะ
    phrûng-níi khɔ̂i dtɔ̀ɔ khâ
    We’ll continue tomorrow.
